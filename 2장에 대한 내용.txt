<p.60>
스크립트 태그의 종류
스크립트 태그		형식			설명
선언문(declaration)	<%! ... %> 		자바 변수나 메소드를 정의하는데 사용한다. <p.63>
스크립틀릿(scriptlet)	<% ... %>		자바 로직 코드를 작성하는데 사용한다. <p.69>
표현문(expression)	<%= ... %>		변수, 계삭식, 메소드 호출 결과를 문자열 형태로 출력하는데 사용한다. <p.71>

====================================================================================================
선언문 태그(declaration) - <%! ... %>
1. 자바 변수나 메소드 등을 선언하는 태그로, 다음과 같은 형식이다.
2. 선언문 태그에 선언된 변수와 메소드는 서블릿 프로그램으로 번역될 때 _jspService()메소드 외부에 배치되므로 JSP 페이지 임의의 위치에서 선언할 수 있다.
3. 심지어 스크립틀릿(scriptlet) - <% ... %> 태그보다 나중에 선언해도 스크립틀릿 태그에서 사용할 수 있다.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
스크립틀릿 태그(scriptlet) - <% ... %>
1. 자바 코드로 이루어진 로직 부분을 표현하며, out 객체를 사용하지 않고도 쉽게 HTML 응답을 만들어낼 수 있다.
2. 가장 일반적으로 사용되며 변수 또는 메소드 호출, 유효식 등 다수를 포함할 수 있다.
3. 각 행이 세미콜론으로 끝나야 한다.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
표현문 태그(expression) - <%= ... %>
1. 웹 브라우저에 출력할 부분을 표현한다.
2. 표현문 태그를 이용하여 선언문 태그 또는 스크립틀릿 태그에서 선언된 변수나 메소드의 반환 값을 외부로 출력할 수 있다.
3. 표현문 태그는 스크립틀릿 태그에서 사용할 수 없으므로 출력할 때는 out.println() 메소드를 사용해야 한다.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
